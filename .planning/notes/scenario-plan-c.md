# 方案C：录制 + AI 增强（后续增强）

在录制模式（方案A）完成后，可选"AI 优化"功能：

- 把录制到的步骤列表 + 当前页面截图一起发给 Anthropic API
- AI 分析截图和步骤，自动补充 wait 步骤（在关键操作后）
- 优化 selector（使用更稳健的选择器）
- 可选：为每个步骤添加中文注释说明
- 入口：录制完成后在步骤列表旁显示"AI 优化"按钮
- 依赖：方案A录制模式 + 方案B的 /api/generate-scenario 端点（可复用）
