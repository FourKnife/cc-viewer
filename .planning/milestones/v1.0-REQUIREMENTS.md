# Cleffa v1.0 — Requirements (Archived)

## Functional Requirements — P0 MVP Core

- [x] **FR-001: 项目启动器** — 用户输入启动命令，后台启动 React 项目 ✅
- [x] **FR-002: 页面嵌入展示** — iframe 嵌入 + 代理跨域 ✅
- [x] **FR-003: 元素选择器** — 悬停高亮 + 点击选中 ✅
- [x] **FR-004: DOM-源码映射** — React fiber _debugSource 定位 ✅
- [x] **FR-005: Chat 对话面板** — 自然语言 prompt + Claude 响应展示 ✅
- [x] **FR-006: Claude 代理通信** — 复用 cc-viewer proxy.js，无需 API Key ✅
- [x] **FR-007: 代码修改应用** — 写入文件 + 热更新 ✅

## Non-Functional Requirements

- [x] **NFR-001: 性能** — 元素选择 < 100ms ✅
- [x] **NFR-002: 兼容性** — Chrome/Edge, React 16+ ✅
- [x] **NFR-003: 安全性** — 仅本地运行 ✅

## Technical Constraints (All Met)

1. ✅ MVP 仅支持 React 项目
2. ✅ 依赖本地 Claude Code
3. ✅ 目标项目需支持热更新
